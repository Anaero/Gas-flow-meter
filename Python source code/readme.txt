Python source code developed using spyder 3.0.1 running with Python 3.5
Pyserial used to support serial comms with Arduino - available via Anaconda opensource platform (see https://www.continuum.io)

Startrun - used to transfer experimental data contained in setup.csv to Arduino at start of run.
Filegrab - transfers eventlog, hourly summary, daily summary and 4-hourly snapshot
Fastgrab - same as filegrab but only transfers eventlog and daily summary
Monitor - simple monitor program to 'eavesdrop' on what's happening
