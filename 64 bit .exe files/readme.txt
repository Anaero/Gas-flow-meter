Windows 10 .exe files created from Python source code using Pyinstaller (see http://www.pyinstaller.org/).  

Note that Pyinstaller is not currently (as of June 2017) compatible with versions of Python above Python 3.5

fastgrab/filegrab V2 - finds path for folder containing executeable and creates /data sub-directory if it doesn't already exist
All log files are then written to the /data sub-directory
